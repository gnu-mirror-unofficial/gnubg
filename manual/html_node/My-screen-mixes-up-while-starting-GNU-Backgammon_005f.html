12.3.4 My screen mixes up while starting GNU Backgammon.

Install the newest driver for your graphic card. Be sure that you have chosen at least a desktop resolution of 65535 (16 bit color depth). If your card is a ATI Rage somewhat there still might be some problems with the graphical interface (GTK+) and MS Windows.
