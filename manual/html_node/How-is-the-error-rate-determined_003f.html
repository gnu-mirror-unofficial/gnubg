12.6.6 How is the error rate determined?

GNU Backgammon determines a player's strength according to it's average error (per move):

Average ErrorSkill Level
0.000 .. 0.005Extra-terrestrial
0.005 .. 0.010World class
0.010 .. 0.015Expert
0.015 .. 0.020Advanced
0.020 .. 0.025Intermediate
0.025 .. 0.030Beginner
0.030 ..Novice

After analyzing a game, match or session you can see the summary in the menu `Analyze, ... statistics'. Note that you can't change the ranges of these values.
